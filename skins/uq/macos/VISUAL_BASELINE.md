# UQ Codex 视觉基线

基线版本：`1.1.1-uq25`

后续修改遵守以下规则：

- 只修改用户指出的区域，不顺带改动其他区域。
- 深色底使用白色文字；白色或浅色底使用 `#24152d` 文字。
- 左侧栏保持 UQ Purple `#51247a`，文字和图标保持白色。
- 顶栏保持 UQ Purple，文字和图标保持白色。
- 普通内容页保持浅色背景，正文卡片保持白底深色字。
- 输入框保持白底深色字；深色按钮保持白色字。
- 白底菜单保持深色字。
- 文件资源卡片保持紫色底、白色字，首次显示时不等待脚本切换颜色。
- 页面变化在下一帧处理，不使用可被连续重置的延迟计时。
- 主题自身的颜色和位置写入不再触发下一轮更新。
- 相同的 class 状态不重复写入 DOM。
- 首页照片、UQ Logo、标题排版保持不变；首页问题标题全部使用白色。
- 已安排任务卡片保持白底深色字。
- 侧栏置顶和归档按钮固定在右侧，标题为按钮预留空间。
- “展开显示”在紫色侧栏中保持白色。
- 主题在 Codex 重启后自动加载。
- 设置页保持紫色左栏、白色 UQ Logo、浅色右栏和 UQ 紫色控件。

每次发布前运行 `tests/run-tests.sh`。视觉基线检查失败时，不安装新版本。
